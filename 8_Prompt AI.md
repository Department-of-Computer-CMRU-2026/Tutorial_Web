## 1. พัฒนาระบบ (Dev)

### 1.1 Admin — ระบบจัดการรายชื่อนักศึกษา (Student Management)

#### Features ที่ต้องพัฒนา
# Amin
| Feature | Prompt AI |
|---|---|
| **CRUD** | ```สร้าง CRUD ระบบจัดการรายชื่อนักศึกษา (Student Management)``` |
| **Switch Alert** | ```เพิ่ม switch alert 2 และ ยืนยันก่อนการลบข้อมูล และเปลี่ยนสถานะ``` |
| **Search** | ```เพิ่ม search ข้อมูลในตารางแบบ Real-time``` |
| **Pagination** | ```เพิ่ม pagination แสดงทีละ 20 แถว``` |
| **import และ Export** | ```เพิ่ม import และ Export ด้วยไฟล์ .csv``` |
| **Check Box** | ```เพิ่ม Check Box เพื่อจัดการข้อมูลแบบทีเดียว``` |
