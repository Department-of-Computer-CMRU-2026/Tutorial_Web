# Ubuntu — คู่มือสอนการใช้งานเบื้องต้นสำหรับมือใหม่

## Ubuntu คืออะไร

**Ubuntu** คือ **ระบบปฏิบัติการ (Operating System)** แบบ Linux

- **Linux** = ระบบปฏิบัติการแบบเปิดต้นฉบับ (Open Source) ใช้กันมากบนเซิร์ฟเวอร์ และบนเครื่องส่วนบุคคล
- **Ubuntu** = แดนหนึ่งของ Linux ที่ใช้งานง่าย มีทั้งแบบ **Desktop** (มีหน้าจอกราฟิก) และแบบ **Server** (ใช้คำสั่งเป็นหลัก)
- ใช้ **Terminal** (หน้าต่างคำสั่ง) เพื่อรันคำสั่ง เช่น ติดตั้งโปรแกรม, จัดการไฟล์, รันสคริปต์

เหมาะกับ: นักพัฒนา, คนที่ใช้เซิร์ฟเวอร์ Linux, คนที่อยากลองใช้ Linux บนเครื่องตัวเอง

---

### พื้นฐานการพิมพ์คำสั่ง

- พิมพ์คำสั่งแล้วกด **Enter** ระบบจะรันคำสั่งนั้น
- เส้นทางโฟลเดอร์และชื่อไฟล์ **ต้องตรงตัวอักษร** (รวมตัวใหญ่-เล็ก บน Linux)
- **พาธ (Path):**
  - **`/`** = รากของระบบ (root)
  - **`~`** = โฟลเดอร์บ้านของผู้ใช้ปัจจุบัน (เช่น `/home/username`)
  - **`.`** = โฟลเดอร์ปัจจุบัน
  - **`..`** = โฟลเดอร์ระดับบน

## การเลื่อนโฟลเดอร์และดูไฟล์

### คำสั่งที่ใช้บ่อย

| คำสั่ง | ความหมาย | ตัวอย่าง |
|--------|-----------|----------|
| **`pwd`** | แสดงโฟลเดอร์ปัจจุบัน (Print Working Directory) | `pwd` |
| **`ls`** | แสดงรายการไฟล์และโฟลเดอร์ในโฟลเดอร์ปัจจุบัน | `ls` |
| **`ls -la`** | แสดงรายการแบบละเอียด รวมไฟล์ซ่อน (ขึ้นต้นด้วย `.`) | `ls -la` |
| **`cd ชื่อโฟลเดอร์`** | เข้าไปในโฟลเดอร์นั้น | `cd Documents` |
| **`cd ..`** | กลับไปโฟลเดอร์ระดับบน | `cd ..` |
| **`cd ~`** | ไปที่โฟลเดอร์บ้าน | `cd ~` |
| **`cd /path/to/folder`** | ไปที่เส้นทางแบบเต็ม | `cd /home/user/Downloads` |

### ตัวอย่างการใช้งาน

```bash
pwd                    # ดูว่าอยู่โฟลเดอร์ไหน
ls                     # ดูว่ามีไฟล์/โฟลเดอร์อะไรบ้าง
cd Documents           # เข้าโฟลเดอร์ Documents
cd ..                  # กลับขึ้นหนึ่งระดับ
cd ~/Downloads         # ไปที่โฟลเดอร์ Downloads ในโฟลเดอร์บ้าน
```

## การสร้าง/ลบ/คัดลอก/ย้ายไฟล์และโฟลเดอร์

### สร้างโฟลเดอร์

```bash
mkdir ชื่อโฟลเดอร์
mkdir project/src      # สร้าง project แล้วสร้าง src ข้างใน (ต้องมี -p ถ้า project ยังไม่มี)
mkdir -p project/src   # สร้างได้แม้ project ยังไม่มี
```

### สร้างไฟล์ว่าง

```bash
touch ชื่อไฟล์
touch readme.txt
```

### คัดลอก (Copy)

```bash
cp ไฟล์ต้นทาง ไฟล์ปลายทาง
cp file.txt file_backup.txt
cp -r โฟลเดอร์ต้นทาง โฟลเดอร์ปลายทาง   # -r = ทั้งโฟลเดอร์ (recursive)
cp -r myfolder myfolder_backup
```

### ย้ายหรือเปลี่ยนชื่อ (Move / Rename)

```bash
mv ต้นทาง ปลายทาง
mv oldname.txt newname.txt    # เปลี่ยนชื่อ
mv file.txt ~/Documents/      # ย้ายไปโฟลเดอร์อื่น
```

### ลบ (Remove)

```bash
rm ชื่อไฟล์                  # ลบไฟล์
rm file.txt
rm -r ชื่อโฟลเดอร์           # ลบโฟลเดอร์และของข้างใน (ระวัง: ลบแล้วกู้ยาก)
rm -rf โฟลเดอร์             # ลบโดยไม่ถามยืนยัน — ใช้ด้วยความระมัดระวัง
```

- **คำเตือน:** `rm -rf` ลบแล้วกู้คืนไม่ได้ ตรวจเส้นทางให้ดีก่อนรัน

### ดูเนื้อหาไฟล์

```bash
cat ชื่อไฟล์                 # แสดงเนื้อหาทั้งไฟล์
cat readme.txt
less ชื่อไฟล์                # แสดงทีละหน้ากด Space เลื่อน กด q ออก
head -n 10 ไฟล์              # แสดง 10 บรรทัดแรก
tail -n 10 ไฟล์              # แสดง 10 บรรทัดท้าย
```

## การแก้ไขไฟล์ข้อความ (nano)

**nano** คือโปรแกรมแก้ไขข้อความใน Terminal ใช้ง่ายสำหรับมือใหม่

### เปิดไฟล์

```bash
nano ชื่อไฟล์
nano readme.txt
```

- ถ้าไฟล์ไม่มี nano จะสร้างไฟล์ใหม่เมื่อบันทึก

### การใช้งานใน nano

| การกระทำ | วิธีทำ |
|----------|--------|
| **เลื่อน cursor** | ลูกศร |
| **บันทึก** | **Ctrl+O** แล้วกด Enter |
| **ออก** | **Ctrl+X** (จะถามบันทึกหรือไม่ถ้ามีการแก้ไข) |
| **ค้นหา** | **Ctrl+W** แล้วพิมพ์คำที่ต้องการหา |
| **ตัดบรรทัด** | **Ctrl+K** |
| **วาง** | **Ctrl+U** |

- ด้านล่างของ nano จะแสดงคำสั่งลัด (^ = Ctrl)

### ตัวแก้ไขตัวอื่น

- **vim** / **vi** — ทรงพลังแต่ต้องเรียนคำสั่ง (กด `i` เพื่อพิมพ์ กด `Esc` แล้วพิมพ์ `:wq` เพื่อบันทึกและออก)
- **gedit** (Desktop) — เปิดจาก Terminal: `gedit ชื่อไฟล์` ถ้ามี GUI

## คำสั่งและเทคนิคที่ใช้บ่อย

### สรุปคำสั่งพื้นฐาน

| การกระทำ | คำสั่ง |
|----------|--------|
| ดูโฟลเดอร์ปัจจุบัน | `pwd` |
| ดูรายการไฟล์ | `ls` หรือ `ls -la` |
| เข้าโฟลเดอร์ | `cd ชื่อโฟลเดอร์` หรือ `cd ..` |
| สร้างโฟลเดอร์ | `mkdir ชื่อโฟลเดอร์` |
| สร้างไฟล์ว่าง | `touch ชื่อไฟล์` |
| คัดลอก | `cp ต้นทาง ปลายทาง` (โฟลเดอร์ใช้ `cp -r`) |
| ย้าย/เปลี่ยนชื่อ | `mv ต้นทาง ปลายทาง` |
| ลบไฟล์ | `rm ชื่อไฟล์` (โฟลเดอร์ใช้ `rm -r`) |
| ดูเนื้อหาไฟล์ | `cat ชื่อไฟล์` หรือ `less ชื่อไฟล์` |
| แก้ไขข้อความ | `nano ชื่อไฟล์` |

### เทคนิคใน Terminal

- **ล้างหน้าจอ:** `clear` หรือ **Ctrl+L**
- **หยุดคำสั่งที่กำลังรัน:** **Ctrl+C**
- **เติมคำสั่งอัตโนมัติ (Tab):** พิมพ์ส่วนหนึ่งของชื่อไฟล์/โฟลเดอร์แล้วกด **Tab** — จะเติมให้หรือแสดงตัวเลือก
- **ประวัติคำสั่ง:** กด **ลูกศรขึ้น** เพื่อเรียกคำสั่งที่เคยใช้
- **ลากไฟล์/โฟลเดอร์ลง Terminal:** บาง Terminal จะใส่เส้นทางให้อัตโนมัติ
